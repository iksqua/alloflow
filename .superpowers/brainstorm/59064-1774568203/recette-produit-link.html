Comment créer le lien Recette → Produit POS ?

La recette est le point de départ. Comment le caissier sait qu'elle est vendable ?

A

Toggle "Vendu en caisse" dans le formulaire recette

Un seul formulaire. Un toggle active la section produit POS (prix de vente, TVA, catégorie caisse). Si désactivé, la recette reste interne.

Pour

  • Un seul endroit pour tout configurer
  • Flow naturel : recette → prix → caisse
  • Impossible d'oublier de créer le produit

Contre

  • Formulaire plus long
B

Flux séparé — lien établi depuis le module Produits

On crée la recette d'un côté, le produit de l'autre. Dans le formulaire produit, un champ "Lier une recette" permet de connecter les deux.

  • Déconnexion entre recette et vente
C

Wizard 2 étapes — recette d'abord, puis "Ajouter au POS ?"

Après avoir sauvegardé une recette, une confirmation apparaît : "Voulez-vous vendre ce produit en caisse ?" Oui → étape rapide de configuration prix/TVA.
