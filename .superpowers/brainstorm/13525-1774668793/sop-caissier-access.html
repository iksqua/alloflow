A — Bouton navbar → Modal plein écran

Bouton "📋 SOPs" dans la barre du haut de la caisse. Ouvre une modale par-dessus le POS avec liste + recherche. Le ticket en cours est préservé en dessous.

Pour

  • Ticket préservé
  • Accès rapide en 1 clic
  • Cohérent avec les autres modals POS

Contre

  • Masque le POS entièrement

B — Panel latéral coulissant

Un panneau glisse depuis la droite par-dessus le ticket. Le POS reste visible à gauche. On peut voir le ticket et les SOPs en même temps.

Pour

  • POS visible en même temps
  • Contexte préservé

Contre

  • Écrasé sur petit écran
  • Masque le ticket sur tablette

C — Page dédiée /caisse/sops

Navigation vers une page séparée dans l'espace caisse. Vue par catégories, mode cuisine intégré. Retour arrière pour revenir au POS.

Pour

  • Plein écran pour lire les étapes
  • Vue catégories claire

Contre

  • Quitte le POS (ticket perdu si non sauvegardé)
  • Friction supplémentaire
