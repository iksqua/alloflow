Principes UX — HOD recommande

🖥️ Interface Caisse

  • Touch targets minimum 48px
  • Max 2 taps pour ajouter un produit
  • Feedback immédiat (animation + son)
  • Zéro erreur possible = confirmations
  • Offline-first, sync auto
  • Landscape par défaut (tablette 10")

📊 Back-office Admin

  • Dense mais lisible — efficacité avant tout
  • Filtres/recherche si > 10 items
  • KPIs visibles sans scroller
  • Bulk actions pour gains de temps
  • Confirmations explicites (soft delete)
  • Responsive tablette 1024×768

📈 Analytics

  • Clarté des données avant esthétique
  • Graphiques simples (barres, courbes)
  • Contexte : comparaison N vs N-1
  • Chaque stat → action suggérée
  • Export rapide (CSV/PDF en 1 clic)
  • Pas de chartjunk (3D, gradients)

Composants UI manquants — À implémenter

Toast System
Feedback non-bloquant après chaque action. Success, error, warning, info. Auto-dismiss 3s.
