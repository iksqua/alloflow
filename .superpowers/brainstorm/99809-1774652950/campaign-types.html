Campagnes — Manuelle ou automatique ?

Ce choix définit la puissance du moteur de campagnes à construire.

A

📤 Campagnes manuelles uniquement

Tu écris un message, tu choisis un segment (ex: "Gold + À risque"), tu envoies. Simple. Promotions flash, annonces, soldes.

Exemples :
"☕ Promo du vendredi — café offert pour tout achat de cookie ce soir !"
"🎄 Fermeture exceptionnelle le 25 décembre — à très bientôt !"
B

⚙️ Campagnes automatiques (triggers)

Le système envoie seul quand une condition est remplie. Zéro action manuelle une fois configuré.

🎂 Anniversaire — SMS automatique J-2 avec code promo
⚠️ Réactivation — "On ne t'a pas vu depuis 30 jours..."
🆕 Bienvenue — SMS après 1ère commande
⭐ Avis Google — Lien envoyé 1h après paiement
👑 Passage VIP — "Tu viens de rejoindre le club Gold !"
C

🔀 Les deux — Recommandé

Campagnes manuelles pour les promos ponctuelles + automations pour les moments clés du cycle client.

Manuel : flash sales, actualités, fermetures, nouveaux produits
Auto : anniversaire, réactivation, bienvenue, Google reviews, upgrade tier
✓ Maximum d'impact avec minimum d'efforts quotidiens
