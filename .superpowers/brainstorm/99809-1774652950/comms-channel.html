Canal de communication — Choix du fournisseur

Ce choix conditionne l'architecture entière. Un seul fournisseur = une seule intégration.

A

🇫🇷 Brevo (ex-Sendinblue) — Recommandé

Plateforme française. SMS + Email + WhatsApp Business dans une seule API. RGPD natif. Parfait pour le marché français.

✓ AVANTAGES
  • Une seule API pour tout
  • Entreprise française (RGPD)
  • WhatsApp inclus
  • SMS ~0.07€/message France
  • Templates visuels
✗ LIMITES
  • WhatsApp : validation Meta requise
  • Plan payant requis (€25/mois)
📦 SMS opérationnel en 24h · WhatsApp en 1-2 semaines (vérification Meta)
B

🌐 Twilio — Puissant mais complexe

Leader mondial. SMS + WhatsApp + Voice. API très flexible. Cher et complexe à configurer pour les non-techniques.

✓ AVANTAGES
  • API très robuste
  • Fiabilité excellente
  • WhatsApp approuvé rapidement
✗ LIMITES
  • SMS plus cher (~0.10€+)
  • Configuration complexe
  • Américain (RGPD à gérer)
  • Pas d'interface visuelle
⚡ Puissant mais surqualifié pour un coffee shop
C

📲 SMS d'abord (Brevo), WhatsApp phase 2

Lancer vite avec SMS uniquement. Ajouter WhatsApp Business une fois que Meta a validé votre compte. Approche itérative.

✓ AVANTAGES
  • MVP rapide (quelques jours)
  • SMS = 98% taux d'ouverture
  • Pas de blocage Meta
  • Tester avant d'investir
✗ LIMITES
  • Pas de rich media (images)
  • WhatsApp décalé dans le temps
🚀 Recommandé si vous voulez lancer dans la semaine

Note WhatsApp Business : Pour envoyer des campagnes sur WhatsApp, Meta exige que votre numéro professionnel soit vérifié (Facebook Business Manager, 1-2 semaines). En attendant, les clients peuvent vous contacter en premier et vous répondez librement. Les campagnes sortantes nécessitent des "templates" pré-approuvés par Meta.
